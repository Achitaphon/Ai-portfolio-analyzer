# 🎓 AI Portfolio Analyzer

AI-powered Portfolio Analyzer สำหรับนักเรียนที่กำลังเตรียมยื่นเข้ามหาวิทยาลัยในระบบ TCAS

## ✨ Features

- วิเคราะห์ Portfolio ด้วย AI
- ประเมินจุดแข็งและจุดที่ควรพัฒนา
- วิเคราะห์ความเหมาะสมกับคณะที่สนใจ
- แนะนำมหาวิทยาลัยที่เหมาะสม
- รองรับการอัปโหลด Portfolio และเอกสารประกอบ
